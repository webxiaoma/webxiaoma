# 面试题前言

前言

在这个网络知识日新月异的年代，让我感受到：

  • 每次找工作时才发现所有的知识点都忘了
  • 每次看面试题试都要从网上查询半天，而且查询的面试题都很散乱，让自己无从复习，而且浪费大量的时间

不知道你是否也有我的痛苦，为了解决我上面的麻烦，我决定汇总一下日常的面试题。
