# 专业性名词

  • LP: (
    LimitedPartner
    )有限合伙人(出钱)。 是项目的主要出资人；
  • GP: (
    GeneralPartner
    )普通合伙人(出力)。负责运营、管理基金的，包括寻找投资机会、管理投资组合公司、设计和实施退出方案等。负有无限的责任
  • SLP（
    SpecialLimitedPartner
    ）为了避S而设立的特殊的LP
  • 天使投资（
    AngelInvestment
    ） 一般以概念式的(或ppt式的产品)获得的初始投资
  • VC:（
    VentureCapital
    ） 风险投资，当企业发展到一定阶段或相对成熟时获得的投资
  • PE:（
    PrivateEquity
    ）私募基金，一般企业发展到后期而获得的投资
  • FOF（
    FundofFund
    ）基金的基金，基金投资基金
  • TOT（
    trustoftrusts
    ） 信托中的信托，专门投资信托产品的信托。
