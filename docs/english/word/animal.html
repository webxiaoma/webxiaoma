# 常见语句与句子

  • endangeredspecies
    /ɛnˈdendʒəd/ /'spiʃiz/ 濒临灭绝的物种
  • Theyareanendangeredspecies
    它们是濒临灭绝的动物
  • looksfierce
    /fɪrs/
    看起来很凶
  • cute
    可爱
  • Thetigerlooksfierce.
    这只老虎看起来很凶
  • Therabbitlooksverycute.
    这只兔子看起来很可爱
  • Kingofthebeasts（/bi:st/）
    兽中之王 `beasts 畜生 兽`
  • Thelionisthekingofbeasts
    狮子是兽中之王
  • Youaresuchabeast
    你真是个畜生
  • inthecage
    在笼子里
  • Thepandaisinthecage
    这只熊猫在笼子里
  • takeapicturewith+人/物
    和谁合影
  • CanItakeapicturewiththelion?
    我可以和这个狮子合影么？
  • takeapictureof+人/物
    是 拍某人/物的照片
  • no-feeding（/'fi:diŋ/）
    禁止喂食
  • There'sano-feedingsign（/saɪn/）overthere.
    那有一个禁止喂食的标志
  • You'rereallyattractedbytheseals，aren'tyou？
    /sil/
    你真的被那些海豹迷住了，不是吗？

    /ə'trækt/
    引起的注意(或兴趣等),吸引;引起;激起
