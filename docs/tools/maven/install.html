# 安装

maven的安装和环境配置文章有很多，可以查看Maven 环境配置 (opens new window)

# IDEA配置Maven

按照好Maven项目并配置好环境变量后，我们就可以在IDEA配置Maven。

# 创建新的Maven项目

在IEDA中创建新的项目可以按照下面步骤

  1. 在IDEA中配置Maven。
  • 设置好Maven的根目录;
  • 设置好xml配置文件路径；
  • 设置好jar包仓库路径
  1. 创建Mave项目
  • 选择创建Maven项目
  • 设置jdk安装路径
  • 选择Maven项目模板

Maven模板常用的有三种cocoon-22-archetype-webapp，maven-archetype-quickstart，maven-archetype-webapp。详细说明戳这里 (opens new window)

  1. 填写项目信息
  • 设置组织ID
  • 设置项目模板id
  • 设置版本号
  1. 设置项目的Mavne信息

archetypeCatalog用来指定maven-archetype-plugin读取archetype-catalog.xml文件的位置：

  • internal 指maven-archetype-plugin内置的
  • local本地的，位置为~/.m2/archetype-catalog.xml
  • remote 指向Maven中央仓库的Catalog
  1. 设置项目名称和地址
  1. 创建后项目模板中没有java目录，我们在main目录中创建java目录，并将java目录设置成源码文件。右击java目录

我们还可以通过右击项目，选择Modules配置项，来配置
