## 前言

`Java` 泛型（`generics`）是 `JDK 5` 中引入的一个新特性, 泛型提供了编译时类型安全检测机制，该机制允许程序员在编译时检测到非法的类型。在`java`集合中，存入对象后，在取出时该对象时，该对象就会成为`Object`类型。这时我们使用泛型就可以避免这种情况。
